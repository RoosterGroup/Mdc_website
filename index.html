⚡ Modern Craft · 生存服务器

欢迎来到 Modern Craft —— 一个以 生存 为核心的 Minecraft 服务器。

📌 版本：Java 版 1.21.11+ ｜ 🔐 密码保护 ｜ 🌍 全球同服 ｜ 📱 多端可登

基本游戏玩法：建筑与红石机器。但是在这个服务器中，你也可以按你自己喜欢的玩法游玩。

你可以建造任何你想建造的东西，比如城市、城镇、村庄或改善现有的城市。

🏛 我们致力于打造 公平、友好、长期 的生存社区。 无论你是建筑大师、红石工程师，还是探险家，这里都有你的一席之地。

  • 📜 你可以看到你的工具或盔甲统计数据，如合成日期、造成的伤害、挖掘的方块、击杀的玩家等。
  • 🖼 网图转地图画。详见Wiki。
  • 💬 发出的消息会短暂的在头顶显示。

基本玩法：建筑、红石。

🔐 登录

使用密码登录保护您的账号安全。即使是正版账号，也需使用密码。

基本命令：

  • /reg <密码> <重复密码> 或 /register <密码> <重复密码> — 注册。两次密码输入必须一样。密码至少6位。
  • /l <密码> 或 /login <密码> — 登录。
  • /unreg — 注销。

🔧 工具使用统计

以下是它统计的所有内容：

  • 开采的方块（使用镐、铲、斧、锄、剪）。
  • 作物被挖掘（用锄头）。
  • 玩家/怪物被杀（使用剑、斧、三叉戟、弓/弩、锤）。
  • 物品在制作、从宝箱/金库/桶中掠夺、交易、通过创造模式生成以及钓鱼获得时的所有权。
  • 受到护甲伤害。
  • 用武器造成的伤害。
  • 鱼已捕到。
  • 绵羊被剪毛了。
  • 射箭（弓/弩）。
  • 鞘翅的飞行时间。

🖼️ 地图画

地图画功能：

  • 从 URL 加载图片到地图或物品展示框 — 支持 PNG、JPEG、WEBP 和 GIF 格式！可以生成地图形式或直接放在物品展示框上（使用智能选择工具选择物品展示框）。自动图片适配意味着图片不会被拉伸！
  • 支持动态 GIF — 如果图片是 GIF，则会动起来！最高可达每秒 20 帧。动画任务异步运行。
  • 自定义宽度和高度 — 让图片横跨多张地图，自动适配所有尺寸，非常适合在物品展示框上展示！
  • 给原版地图添加叠加层 — 手持原版地图创建图像地图，或通过选择物品展示框中的地图来跨多张地图创建。
  • 组合图像地图便于放置 — 将整个多部分图像地图作为一个组合物品获取！自动寻找空间将图像地图放置在物品展示框墙（或地板/天花板）上。点击的物品展示框视为左上角（如果有足够空间）。当被破坏时，整个组合地图会被移除并返回一个组合地图物品。尊重其他插件的物品展示框保护（通过调用 Bukkit 事件）。
  • 轻松刷新地图 — 用一个简单的命令从 URL 源刷新同一图像地图的所有副本！非常适合展示需要定期更新且无需逐个替换的信息。
  • 添加地图标记 — 可使用任何地图图标，比如原版制图师地图上的府邸和神庙图标！还可以在图标下方添加可选文字行。可配置每张地图的最大标记数量。非常适合在自定义图像地图上标记位置！
  • 生存友好 — 要求生存模式的玩家在创建图像地图时，背包中必须有空地图（使用玩家命令）。即使在物品展示框上直接创建图像或获取组合图像地图物品时也有效。可在配置中调整。
  • 易于管理和追踪 — 提供命令帮助玩家和管理员管理自己（或所有人）创建的图像地图。可以与其他玩家共享图像地图的访问权限！可为不同玩家设置不同的权限级别（例如列出某玩家的所有地图、重命名等）。可为每个玩家设置图像创建限制（每个玩家可不同！）。可设置单个图像帧的最大尺寸（同样可自定义）。支持从 ImageOnMap 和 DrMap 迁移（如需更多插件支持请求，请联系我）。
  • 隐形物品展示框 — 内置支持生存友好的隐形物品展示框！用隐身药水喷溅物品展示框即可获得。可在配置中调整。

地图画指令：

  • /imageframe select — 选择要放置图像地图的物品展示框
  • /imageframe create <name> <url> <width> <height> — 创建新的图像地图
  • /imageframe create <name> <url> selection — 创建新的图像地图并直接放入选中的物品展示框
  • /imageframe create <name> <url> <width> <height> combined — 创建新的图像地图并获取其组合图像地图物品
  • /imageframe overlay <name> <url> — 创建新的图像地图，为您手持的原版地图添加叠加层
  • /imageframe overlay <name> <url> selection — 创建新的图像地图，为您选中的物品展示框中的原版地图添加叠加层
  • /imageframe clone <name> <new_name> — 创建与旧地图属性相同的新图像地图
  • /imageframe clone <name> <new_name> selection — 创建与旧地图属性相同的新图像地图并直接放入选中的物品展示框
  • /imageframe clone <name> <new_name> combined — 创建与旧地图属性相同的新图像地图并获取其组合图像地图物品
  • /imageframe playback <name> pause — 切换动态图像地图的暂停状态
  • /imageframe playback <name> jumpto <seconds> — 跳转到动态图像地图的指定秒数
  • /imageframe refresh [optional:image_name] [optional:new_url] — 从源 URL 刷新您创建的地图
  • /imageframe info — 获取您手持的图像地图的相关信息
  • /imageframe get <name> — 获取您自己创建的现有图像地图
  • /imageframe get <name> selection — 将您自己创建的现有图像地图直接放入选中的物品展示框
  • /imageframe get <name> combined — 以组合图像地图形式获取您自己创建的现有图像地图
  • /imageframe delete <name> — 删除您自己创建的现有图像地图
  • /imageframe rename <name> <new_name> — 重命名您自己创建的现有图像地图
  • /imageframe list — 列出您自己创建的所有现有图像地图

📋 服务器规则

服务器宗旨：我们致力于打造一个专注、专业、友好的建筑服务器。请所有玩家遵守以下规则，共同维护优质的创作环境。

1. 核心原则：禁止破坏与抄袭。严禁任何形式的恶意破坏、修改、污染他人建筑作品（包括地形、结构、内饰、红石装置等）。严禁完全照搬或高度模仿其他玩家的原创建筑设计并声称原创。受灵感启发进行创作是允许的，但需保持显著差异并可在交流时注明灵感来源。

2. 合理的资源获取。可以通过远离城市的无人区域挖掘或使用指令 /shop 打开商店购买物品。严禁在他人建筑范围或服务器设施内随意挖掘地形、砍伐森林以获取原材料，破坏自然景观。

3. 良好的交流礼仪。在聊天栏或讨论群中，请保持专业与友善，专注于建筑技巧、红石机械、设计理念的交流。欢迎提出建设性批评，但禁止对他人作品进行人身攻击或恶意贬低。

4. 建筑审核与保护。玩家可向管理员申请对自己的竣工作品进行领地申请，使其免于被他人意外修改。申请成为正式建筑师或将自己的建筑纳入服务器官方展览，需提交作品集（截图/存档）供管理员审核。

5. 违规处理。服务器每周定期备份。对于恶意破坏行为，将采取 【100%回滚操作】，视情况没收物品、暂时或永久封禁等严厉处罚。对于抄袭纠纷，管理员将进行调查与裁定。首次违反可能面临警告、要求修改作品或取消参赛/建筑师认证的处理。

6. 举报与申诉。发现违规行为，请立即截图（含坐标与游戏ID）或录制视频，并向官网 相关链接-反馈/举报 或 QQ群群主 举报。

结语：在 Modern Craft，每一份创意都值得尊重，每一处风景都需共同守护。让我们用方块构建现代美学，用规则奠基创作乐园。
